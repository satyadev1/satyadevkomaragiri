# Komaragiri Satyadev - Interactive Resume

A 3D interactive resume website with a curvy beam timeline, animated background, and smooth scroll.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18 + Vite
- Three.js + React Three Fiber + Drei + Postprocessing
- Framer Motion, GSAP, Lenis
- Tailwind CSS
